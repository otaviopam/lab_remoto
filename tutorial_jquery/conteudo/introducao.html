Basicamente, como se utiliza a jQuery?

O sinal de dólar ($) representa o objeto jQuery (que também é acessível pelo nome jQuery). Através dele acessamos a todas as funcionalidades disponíveis na biblioteca.

jQuery()

A função $() é a principal função da jQuery. Ela retorna uma coleção dos elementos correspondentes ao seletor passado como argumento. Também é possível passar um elemento como argumento.

Seletores: é utilizado o padrão CSS3 (ver seção "Links").

Exemplos:

Código:

<div id="elemento1" class="classe1">ELEMENTO1</div>

<div id="elemento2" class="classe1">ELEMENTO2</div>

Visualização:

ELEMENTO1
ELEMENTO2

$('#elemento1'): Retorna uma coleção com os resultados do seletor utilizado (no caso, um só elemento).

$('.classe1'): Nesse caso, retorna todos os elementos que utilizem a classe 'classe1'.

 

Código:

<div id="pai1">PAI1
    <div id="filho1">FILHO1</div>
    <div id="filho2">FILHO2</div>
    <div id="filho3">FILHO3</div>
    <div id="filho4">FILHO4</div>
</div>
<div id="pai2">PAI2
    <div id="filho1">FILHO1</div>
    <div id="filho2">FILHO2</div>
    <div id="filho3">FILHO3</div>
    <div id="filho4">FILHO4</div>
</div>

Visualização:

PAI1
FILHO1
FILHO2
FILHO3
FILHO4
PAI2
FILHO1
FILHO2
FILHO3
FILHO4

$('#pai1'): Retorna uma coleção com os resultados do seletor utilizado (no caso, o elemento "pai1" e seus respectivos "filhos").

$('#pai1').children(): Retorna uma coleção com os todos os elementos "filhos" do elemento "pai1".

$('#pai1 #filho3'): Nesse caso, retorna uma coleção só com o elemento "filho3".

$('#pai1 #filho3').remove(): Remove o elemento "filho3", "filho" do elemento "pai1".
