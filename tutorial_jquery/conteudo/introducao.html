Basicamente, como se utiliza a jQuery?

Inicialização

Para se inicializar a biblioteca, basta adicionar a seguinte linha na seção <head> da estrutura HTML:

<script src="../caminho/para/jquery.js" type="text/javascript"></script>

$

O sinal de dólar ($) representa o objeto jQuery (que também é acessível pelo nome jQuery). Através dele acessamos a todas as funcionalidades disponíveis na biblioteca.

jQuery()

A função $() é a principal função da jQuery. Ela retorna um "set" dos elementos correspondentes ao seletor passado como argumento. (Também é possível passar um elemento como argumento).

$('div').addClass('minhaClasse');

Princípio:

  • Encontre alguns elementos (utilizando seletores).
  • Faça algo com esses elementos.

Para os seletores é utilizado o padrão CSS3 (ver seção "Links").

Exemplos:

Código:

<div id="elemento1" class="classe1">ELEMENTO1</div>

<div id="elemento2" class="classe1">ELEMENTO2</div>

Visualização:

ELEMENTO1
ELEMENTO2

$('#elemento1'): Retorna um "set" com os resultados do seletor utilizado (no caso, um só elemento).

$('.classe1'): Nesse caso, retorna todos os elementos que utilizem a classe 'classe1'.

 

Código:

<div id="pai1">PAI1
    <div id="filho1">FILHO1</div>
    <div id="filho2">FILHO2</div>
    <div id="filho3">FILHO3</div>
    <div id="filho4">FILHO4</div>
</div>
<div id="pai2">PAI2
    <div id="filho1">FILHO1</div>
    <div id="filho2">FILHO2</div>
    <div id="filho3">FILHO3</div>
    <div id="filho4">FILHO4</div>
</div>

Visualização:

PAI1
FILHO1
FILHO2
FILHO3
FILHO4
PAI2
FILHO1
FILHO2
FILHO3
FILHO4

$('#pai1'): Retorna um "set" com os resultados do seletor utilizado (no caso, o elemento "pai1" e seus respectivos "filhos").

$('#pai1').children(): Retorna um "set" com os todos os elementos "filhos" do elemento "pai1".

$('#pai1 :first-child'): Retorna um "set" só com o primeiro elemento "filho" do elemento "pai1".

$('#pai1 #filho3'): Nesse caso, retorna um "set" só com o elemento "filho3".

$('#pai1 #filho3').remove(): Remove o elemento "filho3", "filho" do elemento "pai1".
