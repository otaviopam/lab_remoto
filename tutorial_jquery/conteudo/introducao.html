Como se basicamente utiliza a jQuery?

O sinal de dólar ($) representa o objeto jQuery (que também é acessível pelo nome jQuery). Através dele acessamos a todas as funcionalidades disponíveis na biblioteca.

jQuery()

A função $() é a principal função da jQuery.

Seletores: é utilizado o padrão CSS3 (ver seção "Links").

Exemplos:
<div id="pai">PAI
    <div id="filho">FILHO</div>
</div>

$('#pai'): Retorna um array com os resultados do seletor utilizado (no caso, um só elemento e seu respectivo "filho").

$('#pai').children('#filho'): Nesse caso, retorna um array só com o elemento "filho".

$('#pai').children('#filho').remove(): Remove o elemento "filho".
