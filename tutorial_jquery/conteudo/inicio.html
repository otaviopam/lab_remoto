O que é jQuery?

jQuery é uma biblioteca Javascript, open-source (MIT), com as seguintes características:

  • Leve: Apenas 32kb (versão para produção)
  • Simples: Fácil de aprender e simples de utilizar/distribuir
  • Compatível: Atende padrão CSS3 e todas as versões de browsers + Internet Explorer 6+ (jQuery 1.x) e Internet Explorer 9+ (jQuery 2.x)
  • Extensível: Arquitetura de plugins flexível e simples
  • Popular: Amplamente utilizada, com farta documentação disponível na Internet
